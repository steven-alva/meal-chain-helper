# 🍱 接龙点餐小助手

把微信群接龙复制进来，小助手帮你生成菜单文案、整理下单清单。

## 怎么打开

如果是从 GitHub 下载，推荐用终端打开：

```bash
cd ~/Desktop
git clone https://github.com/steven-alva/meal-chain-helper.git
cd meal-chain-helper
./start.command
```

以后再打开：

```bash
cd ~/Desktop/meal-chain-helper
./start.command
```

如果已经拿到文件夹，也可以双击这个文件：

```text
start.command
```

或双击中文启动文件：

```text
启动接龙点餐小助手.command
```

如果终端提示仓库不存在或没有权限，需要先把主管的 GitHub 账号加入这个 private 仓库。

### 如果 Mac 提示「Apple 无法验证」

这是苹果对微信/浏览器下载文件的安全提醒，不是小助手坏了。

按下面做一次就好：

1. 先点弹窗里的「完成」，不要点「移到废纸篓」。
2. 按住键盘上的 Control 键，再点「启动接龙点餐小助手.command」，选择「打开」。
3. 如果还是不让开：打开「系统设置」-「隐私与安全性」，在页面下方找到这条拦截提示，点「仍要打开」。
4. 再回到文件夹，重新打开「启动接龙点餐小助手.command」。

第一次启动可能会慢一点，等它自动打开网页就好。

如果窗口提示「没有找到可用的 Python 3.11+」，把这个提示发给安装的人处理一下。

如果网页没有自动弹出，可以手动打开：

```text
http://localhost:8501
```

## 今天怎么用

页面是一个「今日点餐工作台」，按从上到下的顺序来就好。

### 1. 勾选今日菜单

在「今日菜单」里，默认所有菜品都勾选好了。

今天不发的菜，取消勾选即可。

如果有临时菜品，展开「临时加一道菜」添加。

### 2. 生成群文案

确认今日标题后，点击「生成今日群文案」。

下面会出现可复制的群文案，复制到微信群即可。

### 3. 整理接龙清单

接龙结束后，把微信群里的接龙内容贴到「今日接龙」。

点击「整理下单清单」。

复制「主管查看版下单清单」去下单。

## 菜单库维护

平时不用管这个。

只有要更新菜单模板时，展开底部「菜单库维护：导入历史菜单 / 保存模板」。

这里粘贴的是历史菜单，不是今日接龙。

## 需要看一眼的情况

如果页面提示「这些行我有点拿不准」，说明有些接龙没有明确菜品 code，或者格式太自由。

这种行会出现在「需要确认」里，手动看一下就好。

## 用完怎么关

直接关闭启动时弹出的窗口即可。

如果只是关掉网页，小助手还在后台运行；下次可以继续打开网页使用。

## 当前小助手不会做什么

- 不会自动下单。
- 不会自动付款。
- 不会登录外卖平台。
- 不会识别图片。
- 不会记录人员名单。
